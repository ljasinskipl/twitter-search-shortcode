Now you can embed custom twitter search into your blog post. Great tool for live journals or personal notekeeping from conferences.

== Description ==

It's just as simple as that. Enter you custom search for tweets (ex. keyword or hashtag), if you want, include only tweets from one user... There are so many possibilities of use
* personal notekeeping
* conferece live posting
* event's PR
* some more, the only limit is your imagination

== Installation ==

Simple and straightforward.

1. Upload all files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place shortcode [ttdiary] in your post. Detailed info on shortcode arguments on plugin settings page
